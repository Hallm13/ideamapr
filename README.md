# ideamapr
A simple survey tool for complex civic decisions

# Installation

* Install Ruby and Rails
* Download the code
* Run `bundle install`
* Create the database (note that development uses Postgres.)
* Run `rake db:migrate`
* Run `rake db:seed:make_cms_contents`
